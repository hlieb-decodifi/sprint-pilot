Getting Started

  1. Add MCP configuration to your project's .cursor/mcp.json
  2. Copy client scripts (scan-codebase.ts, webhook-receiver.ts)
  3. Start webhook receiver locally: bun run webhook-receiver.ts
  4. Expose with ngrok: ngrok http 3001
  5. Run sprint sync from Cursor IDE
